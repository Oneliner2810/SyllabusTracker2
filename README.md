A beautiful, feature-rich syllabus tracker for SSC exam preparation. Track your progress across subjects, topics, and subtopics with an intuitive 3-level hierarchy.

## ✨ Features

### 📊 3-Level Hierarchy
- **Subjects** (e.g., MATHS, REASONING, ENGLISH)
- **Topics** (e.g., Percentage, Ratio & Proportion)
- **Subtopics** (e.g., Class 1, Class 2, Class 3)

### 🎯 Progress Tracking
- Overall progress pie chart
- Subject-level progress percentage
- Topic-level mini pie charts
- Today's completion counter
- Completion date for each subtopic

### 🎨 UI/UX Features
- 🌙 Dark/Light theme toggle
- 📤 Export data (File download + Clipboard copy)
- 📥 Import data (File select + Paste)
- ↕️ Drag & drop reordering (subjects & topics)
- ✏️ Inline editing (click to edit names)
- 🗑️ Delete confirmation modals
- 🔔 Toast notifications
- 🎉 Confetti celebration on 100% completion
- 📱 Fully responsive design

### 🎨 Visual Indicators
| Progress | Color |
|----------|-------|
| 0% | Orange |
| 1-39% | Orange |
| 40-69% | Yellow |
| 70-99% | Light Green |
| 100% | Green ✓ |

## 🚀 Quick Start

### Option 1: GitHub Pages
1. Fork this repository
2. Go to Settings → Pages
3. Select `main` branch and save
4. Access at `https://yourusername.github.io/ssc-syllabus-tracker`

### Option 2: Local Usage
1. Clone or download this repository
2. Open `index.html` in any modern browser
3. Start adding your subjects!
